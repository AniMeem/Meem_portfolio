ENTERPRISE NETWORK FOR BANKING SERVICES

Created a generic banking network infrastructure using the Cisco packet tracer and real-life network devices. The project aims to create a secure, efficient, interconnected banking network infrastructure. We achieve this by using dynamic routing, VLANs, Inter VLAN Routing, GRE Tunnels, Dynamic Routing(OSPF), and servers with websites, email, DHCP, DNS, etc.

Cisco Routing & Switching VLAN, Inter-VLAN Routing OSPF GRE Tunneling IP Addressing & Subnetting DNS DHCP FTP Email Server Web Server (HTTP) Core Banking System (CBS) iBanking Systems Network Troubleshooting Cisco Packet Tracer

Diabetes Prediction using Classification Algorithms

This research focuses on predicting diabetes in pregnant women using machine learning. The study utilizes various algorithms like SVM, Naive Bayes, Decision Tree, K-nearest Neighbors (KNN), and Logistic Regression. Through the power of these classification algorithms, the goal is to uncover patterns and insights that can aid in predicting diabetes.

Python Machine Learning Classification Algorithms Naive Bayes SVM Decision Tree Logistic Regression KNN Data Analysis
